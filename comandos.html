Las siguientes utilidades son utilizadas en la mayoría de sistemas operativos y permiten diagnosticar problemas, controlar los servicios y cambiar la configuración.

PING

Ping (Packet Internet Groper) es una utilidad que sirve para enviar mensajes a una dirección concreta de red con el fin de realizar una comprobación utilizado en el protocolo ICMP. El nodo destinatario nos responderá con un paquete para confirmarnos que ambos nodos pueden comunicarse perfectamente.

ifconfig / ip a

Permite obtener la información de la configuración IP del nodo, así como modificarla de forma no persistente. El comando iwconfig permite ver la configuración IP de las conexiones inalámbricas.

netstat

netstat proporciona información sobre las conexiones de red.

ip route

ip route muestra la tabla de direccionamiento del nodo en el que se ejecuta.

traceroute

traceroute nos muestra los routers por los que pasa un paquete hasta llegar a su destino.

nslookup

nslookup permite realizar consultas de nombres de dominio y comprobar qué servidor nos ofrece la respuesta.

ifup / ifdown

ambos comandos permiten activar o desactivar la interfaz de red indicada.

Para más información de cada utilidad puede utilizar el comando man seguido de la utilidad de la que se quiere mostrar información.

Por ejemplo: man nslookup
